# 3D Urban Street Racing

A 3D OpenGL-based endless racing game featuring:

- Day/Night Cycle
- AI Overtaking Cars
- Drift Mode
- Boost & Slow Zones
- Jump Mechanics
- Camera Modes (Third, First, Cinematic)
- Health & Speed HUD
- Cheat Mode
- Rear View Camera

## Controls

- A/D → Change Lane
- W/S → Accelerate / Brake
- SPACE → Drift
- V → Change Camera
- N → Day/Night
- B → Rear View
- C → Cheat Mode
- R → Restart
- P → Pause
- ESC → Quit
